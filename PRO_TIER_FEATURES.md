# ClawHQ Pro ($129/mo) — Dashboard Features

Pro includes everything from Starter ($59) plus upgraded tabs and new Pro-only tabs.

---

## Upgraded Tabs (Starter → Pro)

### VPS → Mission Control
- Reboot VPS
- CPU/RAM/disk/network usage graphs over time
- Process list
- Network stats
- Scheduled restarts
- Maintenance windows

### Models (Advanced)
- Run multiple models simultaneously
- Model playground — test prompts side-by-side before deploying
- Custom model config — temperature, top-p, system prompts per model

### Agents (Advanced)
- No-code agent builder from dashboard
- Deep usage analytics — conversations/day, response times, user satisfaction
- Workflow automation — chain agents together, trigger-based actions

### Channels (Advanced)
- Channel analytics — messages per channel, response rates, peak times
- Auto-responses — fallback messages, business hours, away messages

### Monitoring (Advanced)
- Real-time dashboards with charts (requests/min, response latency, uptime history)
- Alerts for downtime, high CPU, agent errors (email/webhook)
